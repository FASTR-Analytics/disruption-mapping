# Exporting Maps

## Download as PNG

Each map view has a **Download Map as PNG** button.

### Single Indicator Map

1. Configure your map (year, indicator, labels)
2. Click **"Download Map as PNG"**
3. The file downloads with naming format: `{country}_{indicator}_{year}_{date}.png`

### Multi-Indicator Map

1. Select your 2 indicators
2. Click **"Download Multi-Indicator Map"**
3. The file downloads with naming format: `{country}_multi_indicator_{year}_{date}.png`

## Image Specifications

| Property | Single Map | Multi-Indicator |
|----------|------------|-----------------|
| Width | 12 inches | 14 inches |
| Height | 10 inches | 10 inches |
| Resolution | 300 DPI | 300 DPI |
| Format | PNG | PNG |

## Map Elements

Exported maps include:

- **Title**: Indicator name (and country for multi-indicator)
- **Subtitle**: Time period
- **Legend**: Color scale with percentage labels
- **Scale bar**: Distance reference
- **North arrow**: Orientation indicator
- **Caption**: Color interpretation guide

## Tips for Reports

!!! tip "Presentation Quality"
    The 300 DPI resolution ensures maps are crisp when printed or displayed in presentations.

!!! tip "Consistent Styling"
    Maps use consistent styling across all exports for professional reports.

## Troubleshooting

### Download not starting

- Check browser popup blocker settings
- Try a different browser
- Ensure the map has fully loaded before downloading

### Map appears blank

- Wait for the map to finish rendering
- Check that data exists for the selected parameters
